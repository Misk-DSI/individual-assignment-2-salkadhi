# Red Wine Dataset Analysis

This repo contains and R Markdown document, where I provide an analysis of the red wine dataset from kaggle. The wine variety is “Vinho Verde” from Portugal. The dataset describes physiochemical properties of the wine samples taken at the certification stage plus an expert assessment of quality.

Go the the gh-pages branch of this repo ('Misk-DSI/individual-assignment-2-salkadhi') [here](https://Misk-DSI.github.io/individual-assignment-2-salkadhi/), which has is the source for the accompanying GitHub pages site.
